

This is a fullstack project setup with a Vite + React frontend and an Express backend. 

## Project Structure

- `/frontend`: Vite + React frontend. Uses modern CSS aesthetics with Google Fonts (Inter & Outfit).
- `/backend`: Node.js + Express backend.

## How to Run

You will need two separate terminal windows.

### 1. Start the Backend

```bash
cd backend
npm start
```
This will start the backend server on port `5000`.

### 2. Start the Frontend

```bash
cd frontend
npm run dev
```
This will start the Vite frontend server. Open the URL provided in the terminal (usually `http://localhost:5173`) in your browser.

## Features Included
- Beautiful, modern UI with dynamic gradients and glassmorphism cards.
- Real-time connection status between frontend and backend.
- Ready-to-go Express API (`/api/hello`).
